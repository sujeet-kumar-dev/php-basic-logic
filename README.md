# PHP Basic Logic Practice

This repository contains basic PHP logic practice programs to strengthen core programming fundamentals.

## Topics Covered

- Even / Odd number check
- Fibonacci series
- Factorial of a number
- Prime number check
- Palindrome number
- Sum of digits
- Reverse number
- Array operations (max, min, sorting)
- String manipulation (reverse, vowel count)

## Purpose

The goal of this repository is to improve logical thinking and strengthen PHP fundamentals for backend development and technical interviews.
